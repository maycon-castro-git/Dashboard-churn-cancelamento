# Dashboard-churn-cancelamento
DASHBOARD FINANCEIRO

📊 Análise de Churn Financeiro e Receita Perdida
Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o churn financeiro de uma empresa de telecomunicações, identificando padrões de cancelamento, impacto na receita e oportunidades de retenção de clientes.

Mais do que construir um dashboard, o foco foi transformar dados brutos em informações estratégicas para apoiar a tomada de decisão.

Problema de Negócio

A empresa possuía um grande volume de cancelamentos, porém sem uma visão consolidada que permitisse entender:

Quais motivos geram mais cancelamentos;
Qual o impacto financeiro das perdas;
Quais planos possuem maior risco de churn;
Quais regiões demandam maior atenção;
Como priorizar ações de retenção.

Sem essa análise, as decisões eram tomadas com pouca visibilidade sobre os fatores que realmente impactavam a receita.

Objetivo

Desenvolver uma solução analítica capaz de:

Monitorar cancelamentos;
Quantificar receita perdida;
Identificar padrões de churn;
Apoiar estratégias de retenção;
Gerar insights para redução de perdas financeiras.

Processo Analítico
1. Extração dos Dados

Os dados foram consultados diretamente via MySQL utilizando consultas SQL para obtenção das informações de clientes, planos, cancelamentos e faturamento.

2. Tratamento dos Dados
Limpeza de registros inconsistentes;
Padronização de campos;
Tratamento de valores nulos;
Criação de métricas de negócio.
3. Modelagem

Construção do modelo relacional para análise dos indicadores de churn e receita perdida.

4. Visualização

Desenvolvimento do dashboard interativo no Power BI com filtros por:

Principais Insights
Inadimplência

A inadimplência representa o principal motivo de cancelamento.

Impacto: Perda significativa de receita recorrente.

Ação sugerida: Estratégias preventivas de cobrança e recuperação.

Concentração Geográfica

São Paulo concentra a maior parte dos cancelamentos registrados.

Ação sugerida: Investigar fatores regionais e qualidade do serviço.

Planos com Maior Impacto Financeiro

Os planos de maior velocidade concentram grande parte da receita perdida.

Ação sugerida: Campanhas de retenção direcionadas e revisão de benefícios.

Tempo Médio de Permanência

Os clientes permanecem em média 458 dias antes do cancelamento.

Ação sugerida: Identificar sinais de risco antes da saída do cliente.

Dashboard

O dashboard permite:

Monitorar churn financeiro;
Analisar receita perdida;
Avaliar desempenho por plano;
Investigar causas de cancelamento;
Apoiar decisões estratégicas.
